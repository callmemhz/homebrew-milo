# homebrew-milo (RETIRED)

This tap was retired in 2026-07 when callmemhz/milo went private —
the formula download URLs no longer work.

Install milo via the internal installer instead (requires `gh` logged in
to an account with access to ApodexAI/milo):

```bash
gh api repos/ApodexAI/milo/contents/install.sh -H "Accept: application/vnd.github.raw" | bash
```
